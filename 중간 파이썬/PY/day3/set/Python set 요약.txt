A = {'돈까스','보쌈','제육덮밥'}
B = {'짬뽕','초밥','제육덮밥'}

print(A.intersection(B))
=> {'제육덮밥'}
A 와 B가 공통으로 좋아하는 음식 (교집합)
-----------------------------------------------------
print(A.union(B))
=>{'돈까스','보쌈',짬뽕','초밥','제육덮밥'}
------------------------------------------------------
print(A.difference(B))
=>{'짬뽕','초밥','제육덮밥'}
 
A만 좋아하는 음식 (차집합) => {'돈까스','보쌈'}
